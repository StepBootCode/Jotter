# Jotter

Структура базы данных

Таблица NOTES
	id - уникальный идентификатор записи
	date - Датаи время записи
	name - Наименование записи
	note - сама запись
	img_id - Ссылка на картинку или null
	check - Истина если есть срок
	Crypto - Истина если шифруется/требуется пароль
	todate- срок исполнения

Таблица IMG
	id - уникальный идентификатор записи
	image - линк на картинку в асетах
